## 🛠 사용 기술

- **Engine:** Unity 2022.3.62f3
- **Language:** C#
- **개발 인원:** 1인 (Solo Development)
- **개발 기간:** 2025.11 ~ 2026.03

---

## ⚙️ 주요 기능

| 기능 | 설명 |
|------|------|
| 드래그 앤 드롭 인벤토리 | 아이템 이동, 슬롯 타입별 장착 제한 |
| 쿼터뷰 조준 시스템 | 반동 + 카메라 각도 보정 |
| NavMesh 적 AI | 상태 머신 (Patrol → Chase → Attack) |
| 보안 슬롯 | 사망해도 아이템 유지 |
